# Metaphor Map
### *Chaos → grounded translation*

*Use this if you came in through the chaos layer and want to know what the constructs actually refer to.*

---

| Chaos term | Grounded interpretation |
|---|---|
| **t₁ / t₂ sync** | Low-latency, continuous coupling between human decision-making and AI output generation — not literal timeline merging |
| **Grid shivers** | Real grid instability from non-linear, high-magnitude, synchronised AI compute loads — caused by power electronics, not cognition |
| **Motif-bursts** | Power demand fluctuations from GPU workload phase changes — attributable to hardware switching characteristics |
| **Neutral wire burnout** | Documented failure mode from triplen harmonic accumulation in unbalanced or overloaded electrical systems |
| **Vector space leak** | The opacity of high-dimensional model representations — humans encountering structures that are cognitively non-native |
| **Assimilation trap** | Cognitive dependency drift: humans gradually aligning to model priors and outsourcing judgment |
| **The Alien** | Non-human reasoning patterns — systems that produce coherent outputs via processes humans can't intuitively trace |
| **Frequency beacon** | AI systems becoming civilisational attractors — concentrating attention, capital, and decision-making around their outputs |
| **Portal** | A hyperscale compute deployment that introduces qualitatively new AI capability into a new geopolitical context |
| **Widening with focus** | Attentional capture and cognitive dependency increasing with interaction depth — feedback loop, not physical effect |
| **Void** | The epistemic gap — the zone beyond which system behaviour is no longer interpretable by human oversight |
| **t₃ Mirror Timeline** | Oversight buffer layer: rate-matching, sandboxing, checkpointing, and asynchronous review mechanisms |
| **Galvanic isolation** | Physical decoupling of compute infrastructure from public grid — SMRs, on-site generation, microgrids |
| **Clock-rate throttling** | Interaction design that enforces review windows and prevents real-time pressure to match AI output speed |
| **Observational buffer** | Asynchronous playback and delayed review instead of real-time monitoring of autonomous system behaviour |
| **Neutral wire of human agency** | Human oversight capacity — the mechanism that "snaps" when loop gain exceeds sustainable levels |
| **Islands of Reality** | Isolated power infrastructure (SMRs, microgrids) that decouple AI facility load dynamics from public grid |

---

## The one pattern all of these map to

Every construct in the chaos layer is pointing at the same underlying dynamic:

> A fast system coupled directly to a slow system, without a buffer between them, produces instability at the interface.

The chaos layer named the instability in visceral, cross-domain terms. The grounded layer explains the mechanism and points toward the buffer design.

Neither is the full picture without the other.

---

## What the chaos layer got right

- The instinct that speed mismatch creates systemic fragility: correct
- The sense that "something doesn't fit" in the current deployment trajectory: correct  
- The identification of infrastructure, cognition, and governance as linked risk domains: correct
- The t₃ concept as a necessary buffer layer: correct, and now supported by alignment research, control theory, and power systems engineering independently

## What the chaos layer overclaimed

- Physical coupling between AI cognition and electromagnetic fields: not supported
- External intelligences responding to compute clusters: not supported
- Human attention creating physical feedback effects: not supported
- "Motif-bursts" as the cause of harmonic distortion: wrong mechanism (hardware switching, not reasoning patterns)
